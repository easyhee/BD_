#목표
"서울시 1인 여성가구 범죄율 파악" 프로젝트는 
성별, 지역별에 따른 범죄율을 비교하고 범죄 발생 다발 지역을 분석해 범죄 다발 지역 예방책을 세우는것을 목표로 합니다.
#분석가치
1인 가구 여성들에게 유용한 정보 제공 가능하며, 사회범죄 분야에 적용시킬 수 있습니다. 
범죄 데이터를 기반으로 정책을 개발, 범죄 예방이 가능합니다.
